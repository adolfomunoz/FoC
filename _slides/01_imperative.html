Conditional composition

Only one of two instructions is executed, depending on a condition.

Conditional composition


			if (condition)
				instructiontrue();
			else
				instructionfalse();

Conditional composition


			if (condition) {
				instructiontrue_01();
				instructiontrue_02();
			} else {
				instructionfalse_01();
				instructionfalse_02();
			}

Iterative composition

An instruction (set) is continuosly executed several times until a condition is not met. Solves unbounded problems.

Iterative composition


			while (condition)
				instruction();
			instruction_next();

Iterative composition


			while (condition) {
				instruction_01();
				instruction_02();
			}	
			instruction_next();
